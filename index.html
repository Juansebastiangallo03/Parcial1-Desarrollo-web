HTML

Lenguaje de marcado con el que declaramos la estructura de una página web. Consiste de elementos anidados que permiten modificar su comportamiento.

DNS

Nos permite hablar con los servidores en internet sin necesidad de aprendernos sus direcciones IP.

TCP/IP

Es un conjunto de protocolos que permiten la comunicación entre dispositivos informáticos en una red, incluyendo Internet.

CSS

Lenguaje de estilos que permite definir cómo se verá una página web. Controla la presentación de los elementos estructurados en HTML.

Página sobre HTML

HTML es el lenguaje de marcado que se usa para crear y estructurar las páginas web.

Página sobre CSS

CSS es el lenguaje de estilos usado para controlar la apariencia de las páginas web.
